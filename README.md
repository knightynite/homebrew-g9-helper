# Homebrew Tap for G9 Helper

[G9 Helper](https://github.com/knightynite/HiDPIVirtualDisplay) — free macOS menu bar app for HiDPI scaling on ultrawide monitors.

## Install

```bash
brew install knightynite/g9-helper/g9-helper
```

## Update

```bash
brew upgrade g9-helper
```

## Uninstall

```bash
brew uninstall g9-helper
```
